Antifer Geometry Calculator

Standalone HTML application for Antifer geometry calculation, closed B-rep generation, topological validation, volume validation and in-browser 3D visualisation. The model is generated directly from dimensional coefficients, with no embedded template solid, no external JavaScript library, no CAD viewer and no server requirement.

Inputs

The solid is generated directly from coefficients. Displayed and exported coordinates shift X and Y to the homogeneous centre of mass while preserving the physical bottom datum at z = 0.

Drawings are generated in-browser from the same active dimensional coefficients used by the solid model. Top and bottom plans use the current groove, radius and chamfer geometry; the front view shows the A/B/H tapered elevation.

OBJ, STL and the calculation report are generated from the same closed B-rep used by the viewer. The plain-text report remains available through the export buttons. STL is an ASCII triangular surface mesh; OBJ uses indexed triangular faces. Coordinates are in metres.
